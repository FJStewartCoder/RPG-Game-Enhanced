# RPenGine
## Introduction
RPenGine is a text-based game engine designed for creating basic RPG style games.

Documentation can be found [here](/docs).

A campaign by the name of "Sample" is included with the engine. This shows basics of what is possible.  
**this is not integral to the function of the engine and can be deleted*

Thank you for using the engine.  
*don't delete the license; you have been warned*

## Compiling
### Windows
The engine requires Lua 5.4 or lower to function as intended. Install suitable Lua development binaries and replace [include/lua](include/lua) with the include directory.  
Replace the library in [lib](lib) with the one provided and include the .dll

### Linux
Run the Makefile.
